## 📌 Project Description

This project is an audit of the Python programming language as part of the Open Source Software course. It explores the origin, philosophy, licensing, Linux integration, ecosystem, and comparison with proprietary alternatives.

The project also includes five shell scripts demonstrating basic Linux and automation skills.

---

## 💻 Scripts Included

### 1. System Identity Report

Displays system information such as kernel version, user, uptime, date, and Linux distribution.

### 2. FOSS Package Inspector

Checks whether Python is installed, shows its version, and prints a short description.

### 3. Disk and Permission Auditor

Analyzes important system directories and displays their size, permissions, and ownership.

### 4. Log File Analyzer

Reads a log file, counts occurrences of a keyword (like "error"), and displays results.

### 5. Open Source Manifesto Generator

Takes user input and generates a personalized open-source philosophy statement.
